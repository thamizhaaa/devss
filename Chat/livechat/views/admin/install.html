Configuration

Welcome to PHP LiveChat. Before getting started, application needs to be configured to play along with your database server. You will need the following information during installation (if you don't know those items, you can get them from your web hosting company):

1. Database host
2. Database port number (only if other than the default one)
3. Database name
4. Database username
5. Database password

If the given database doesn't exist yet, the installation wizard will try to automatically create it for you. On some servers, however, this attempt will fail.
