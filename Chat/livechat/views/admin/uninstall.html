Uninstallation

This is the uninstallation page. If you no longer wish to use the application, continue by clicking the Uninstall button.

Uninstalling means that operator accounts, messages history and canned messages will be removed. The administrator account, however, will remain available and you can sign in again later. Database tables will be removed, but the database itself will still exist (you might want to remove it manually, if needed).

After uninstallation process is done, all application's data (operator accounts, messages history, canned messages) will be lost. If necessary, before uninstalling, you can backup application database tables using your favorite database administration tool (e. g. phpMyAdmin).
